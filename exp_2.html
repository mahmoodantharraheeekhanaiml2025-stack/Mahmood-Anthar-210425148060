String Matching

Three ways to find a pattern inside text. Naive brute-forces every alignment. KMP remembers the pattern's own structure so it never re-checks a character it already matched. Rabin-Karp hashes the pattern and slides a rolling hash across the text, only comparing characters when hashes collide.
